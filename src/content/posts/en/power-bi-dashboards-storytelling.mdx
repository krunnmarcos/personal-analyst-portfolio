# Power BI: tips for dashboards that tell stories

A beautiful dashboard isn't necessarily a useful dashboard. After creating dozens of reports in Power BI, I learned that **clarity beats complexity** — always.

## 1. Start with the question, not the chart

Before dragging any visual, ask: *"What decision does this person need to make?"*. This completely changes the approach.

## 2. Less is more

My personal rule: at most **5 visuals** per page. If you need more, create another page. White space is your ally.

## 3. Use color with intention

Color isn't decoration — it's information. I use red only for alerts, green for goals achieved, and gray for everything that's context.

## 4. Titles that explain

Instead of "Sales by Month", use "Sales grew 12% compared to the previous month". The title **is already the insight**.

## Result

By applying these techniques, stakeholder engagement with the reports increased significantly. Today, managers check the dashboards daily — before, they barely opened the weekly link.
