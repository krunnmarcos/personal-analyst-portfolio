# Power BI: dicas para dashboards que contam histórias

Um dashboard bonito não é necessariamente um dashboard útil. Depois de criar dezenas de relatórios no Power BI, aprendi que **clareza vence complexidade** — sempre.

## 1. Comece pela pergunta, não pelo gráfico

Antes de arrastar qualquer visual, pergunte: *"Que decisão essa pessoa precisa tomar?"*. Isso muda completamente a abordagem.

## 2. Menos é mais

Minha regra pessoal: no máximo **5 visuais** por página. Se precisa de mais, crie outra página. Espaço em branco é seu aliado.

## 3. Use cores com intenção

Cor não é decoração — é informação. Uso vermelho apenas para alertas, verde para metas atingidas, e cinza para tudo que é contexto.

## 4. Títulos que explicam

Em vez de "Vendas por Mês", use "Vendas cresceram 12% em relação ao mês anterior". O título **já é o insight**.

## Resultado

Aplicando essas técnicas, o engajamento dos stakeholders com os relatórios aumentou significativamente. Hoje, os gestores consultam os dashboards diariamente — antes, mal abriam o link semanal.
